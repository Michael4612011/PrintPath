# PrintPath

PrintPath is an open-source print path planning engine for FDM 3D printers.

The project explores smarter algorithms for print order optimization, travel path reduction, and G-code analysis while remaining compatible with existing slicers.

## Why?

PrintPath started with a simple question:

> Why does a slicer sometimes choose a print path that seems less logical than an obvious alternative?

Instead of accepting the default behavior, this project aims to study, understand, and improve print path planning algorithms.

## Goals

- Optimize travel moves
- Improve island ordering
- Reduce unnecessary movements
- Preserve print quality
- Stay compatible with existing G-code
